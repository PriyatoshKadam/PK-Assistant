# PK-Assistant

PK-Assistant is a virtual assistant application that provides basic functionalities to interact with users through text commands.

## Features
- Submit commands via text input
- Receive responses from the virtual assistant
- Basic commands supported: playing music, checking the time, and searching for information

## Installation
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/your_username/pk-assistant.git
   ```
2. Navigate to the project directory:
   ```bash
   cd pk-assistant
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Run the Flask application:
   ```bash
   python app.py
   ```
2. Open your web browser and navigate to http://localhost:5000.
3. Enter a command into the text input field and click the "Submit" button.
4. The virtual assistant will process your command and provide a response.
